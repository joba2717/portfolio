SuperQuiz

Some projects basically had no instructions other than "create something". The final project in a course about database technology was just like that. I was set to create a web application utilizing a SQL database but other than that, NO INSTRUCTIONS. So, me and my partner came up with Super Quiz, a fun trivia site where users can test their trivia skills in a variety of categories.

Developing Super Quiz was a lot of fun. The quiz site is simple, easy to use and kind of addicting if you like trivia knowledge. For the questions we used an open trivia database with over 3700 questions. We integrated the trivia database to our system using the API connected to the trivia database. We wrote the application in Visual Studio using .NET Core MVC.

Creating a working web application with an integrated SQL database helped me learn a lot about the basics of web development as well as SQL in general.
